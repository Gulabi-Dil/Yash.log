# Yash.log

My personal blog where I document things I build, break and learn along the way.

Initially intended to go full MERN for this but seeing how it won't be much of an active webssite, I opted for static approach. The blogs are written in markdown format with frontmatter.

## Features:

- Automatic post loading from `/blogs`.
- Clean newspaper-style UI
- Tag system for posts.

More features coming soon!

Any suggestions are welcome!
